What is binary bearch?

Binary search is an algorithm that finds the index position of a target number within a sorted array of numbers.


How does binary search work?

In order to search for the target, it splits the array it's searching in half in order to identify whether the target number is bigger or smaller than the probe number.


Why binary search?

The benefit of this algorithm is its speed at elimitating which half the target number is located because at its core, it splits the array in which it's searching in half in order to idenify whether the target number is bigger or smaller than the probe number.
