What is binary search?

Binary search is an algorithm that finds the index position of a target number within a sorted array of numbers.


How does binary search work?

To locate the target number, a binary search takes a divide and conquer approach: We identify the number that is located in the middle of the sorted array and compare to see if it is smaller than, bigger than, or the same as the target number we're trying to locate.

Why binary search?

The benefit of this algorithm is its speed at elimitating which half the target number is located because its worse-case time complexity dependent on the input (in our case the length of the array), is goin to be logarithmic O(log n). B-search takes a divide and conquer strategy by splitting the array it is searching in half and checking to see if the number located in the dead center of the array on that iterartion of the algorithm is bigger, smaller, or the same as the target number we are trying to locate.
